== README

This is an API for the University of Rochester project WeBWork and a subproject of Grapefruit, an RCOS project at RPI. Grapefruit's github repository can be found at https://github.com/grahamcracker/grapefruit

Goals:
	-When a new user is created on Grapefruit, the API will create a new user on WeBWork
	-When a user logins, the API should log them into WeBWork
	-Grapefruit shold have access to information such as whether or not specific problem sets have been comoleted
